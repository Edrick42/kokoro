# Building Kokoro — Learn Rust by Creating a Living Creature Game

> From zero Rust knowledge to a complete game with AI, genetics, and persistence.

## About This Book

This book teaches the Rust programming language from absolute scratch. Instead of isolated exercises, every concept is taught through building **Kokoro** — a virtual creature game where digital beings have unique genomes, emotions, learning neural networks, and persistent memories.

By the end, the reader will:
- Master Rust's ownership model, type system, traits, generics, lifetimes, and error handling
- Understand ECS (Entity Component System) architecture and game development patterns
- Build a complete, working game with AI, genetics, persistence, and modular visual systems
- Have production-quality code they can extend, publish, or use as a portfolio piece

## Book Structure

The book is organized in **4 Parts**, progressing from foundational Rust to advanced systems:

### Part I — Foundations: Rust from Zero
Covers the language itself. Each chapter introduces concepts with small examples first, then applies them to the Kokoro project.

### Part II — Building the Game Core
Builds the creature's genome, mind, and world systems. Teaches Bevy ECS, state machines, and game loops.

### Part III — Intelligence and Evolution
Implements neural networks, genetic crossover, and multi-creature management. Covers advanced Rust patterns.

### Part IV — Polish and Beyond
UI, visual effects, persistence, testing, and preparing the project for distribution.

## How to Read This Book

Each chapter follows a consistent structure:
1. **Concept** — The Rust concept explained clearly with minimal examples
2. **In Practice** — How that concept appears in Kokoro's real code
3. **Build It** — Guided implementation where the reader writes the code
4. **Deep Dive** — Why Rust works this way (for curious readers)
5. **Checkpoint** — What the project should look like after this chapter

Code examples reference actual files in the `src/` directory so the reader can cross-reference the full codebase.

## Prerequisites

- Basic programming knowledge (any language — Python, JavaScript, Java, etc.)
- A computer with internet access
- No prior Rust or game development experience needed

## Project Repository

The complete source code is available alongside this book in the `src/` directory. Each chapter's checkpoint is tagged in git so the reader can jump to any point.
