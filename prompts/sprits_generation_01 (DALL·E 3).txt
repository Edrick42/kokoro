# Kokoro — DALL·E 3 Prompts (ChatGPT)

## How to use

1. Start a NEW ChatGPT conversation
2. Send **Prompt 0** (Idle) first — no image attached yet
3. Pick the result you like most
4. For every prompt after that, **attach the Idle image** to the message before sending
5. Go through prompts 1–7 in order
6. At the end, do the color variants (section below)

## Saving and naming your files

Once you have all sprites, save them as:

```
assets/sprites/kobara/
├── idle_0.png
├── happy_0.png
├── hungry_0.png
├── tired_0.png
├── sleeping_0.png
├── lonely_0.png
├── playful_0.png
└── sick_0.png
```

If you generate multiple frames of the same mood (for animation), name them:
`idle_0.png`, `idle_1.png`, `idle_2.png`, etc.

## If the character drifts

DALL·E sometimes shifts the design after several generations. If that happens:

1. Go back to your favorite Idle result
2. Start a fresh ChatGPT conversation
3. Attach the Idle image as the first message
4. Say: "This is my character reference. Do not change its design.
   I will now ask for pose variations."
5. Then continue from whichever mood you need
